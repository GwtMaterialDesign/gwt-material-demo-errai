Usage

Using MaterialIcon Widget you can easily make it by adding the lines below. Just set the icon attribute and indicate which icon you want to display. Please refer below about the list of icon.

3d_rotation 
<m:MaterialIcon iconType="ROTATION_3D" />
Sizes

To control the size of the icon, just set size attribute in your MaterialIcon widget.

polymerTiny
polymerSmall
polymerMedium
polymerLarge 
 <m:MaterialIcon iconType="POLYMER" iconSize="TINY" />
 <m:MaterialIcon iconType="POLYMER" iconSize="SMALL" />
 <m:MaterialIcon iconType="POLYMER" iconSize="MEDIUM" />
 <m:MaterialIcon iconType="POLYMER" iconSize="LARGE" />
Icon inside MaterialLink

You can add in your material link an icon instance by adding icon attribute

favoriteI love Material Design 
<m:MaterialLink fontSize="2em" text="I love Material Design" textColor="red" iconType="FAVORITE" iconPosition="LEFT" />
