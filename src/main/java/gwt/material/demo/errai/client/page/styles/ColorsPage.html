Usage

Here is a backgroundColor palette based on the material design base backgroundColors. Each of these backgroundColors is defined with a base backgroundColor class and an optional lighten or darken class.

Background backgroundColor
To apply a background backgroundColor, just add the backgroundColor name and light/darkness as a class to the element.

This is a Material Panel with a TEAL_LIGHTEN_2 class

 <m:MaterialPanel backgroundColor="TEAL_LIGHTEN_2">  <m:MaterialLabel text="This is a Material Panel with a TEAL_LIGHTEN_2 class" /> </m:MaterialPanel>

Text backgroundColor
To apply a text backgroundColor, just append -text to the backgroundColor class like this:

This is a Material Panel with dark BLUE text

 <m:MaterialPanel backgroundColor="BLUE-text text-darken-2" textAlign="CENTER">  <m:MaterialLabel text="This is a Material Panel with dark BLUE text" /> <m:MaterialPanel>
